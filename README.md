# AF05149694--MedicineTracker
A Medicine Tracking System using JDBC and MySQL is a Java-based application that manages medicine inventory. It performs CRUD operations to add, view, update, and delete records. It tracks quantity, price, and expiry dates, helping pharmacies maintain accurate stock and avoid selling expired medicines.

🩺 Medicine Tracker System
📌 Description
The Medicine Tracker System is a Java-based application developed using Eclipse IDE that helps users manage and track their daily medicines efficiently. It is designed to ensure users never miss their doses and can maintain a proper record of their medication schedule.
🚀 Features
Add, update, and delete medicine details
Set dosage timings and reminders
Track daily medicine intake
View medicine history and schedule
Simple and user-friendly interface
🛠️ Technologies Used
Java (Core Java, OOP concepts)
Eclipse IDE
💡 Use Case
This project is useful for patients, elderly people, or anyone who needs to manage multiple medicines regularly.
▶️ How to Run
Open the project in Eclipse IDE
Compile and run the main Java file
Follow the console/interface instructions
