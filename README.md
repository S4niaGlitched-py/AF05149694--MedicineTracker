# AF05149694--MedicineTracker
A Medicine Tracking System using JDBC and MySQL is a Java-based application that manages medicine inventory. It performs CRUD operations to add, view, update, and delete records. It tracks quantity, price, and expiry dates, helping pharmacies maintain accurate stock and avoid selling expired medicines.

# 🩺 Medicine Tracker System

## 📌 Description
The Medicine Tracker System is a Java-based application developed using Eclipse IDE. It helps users manage and track their daily medicines so they never miss a dose and can maintain a proper schedule.

## 🚀 Features
- Add medicine details
- Update medicine information
- Delete medicine records
- Set dosage timings
- Track daily medicine intake
- View medicine schedule/history
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Java (Core Java, OOP Concepts)
- Eclipse IDE
- File Handling / Database (if used)

## 💡 Use Case
Useful for patients, elderly people, or anyone who needs to take medicines regularly and wants proper tracking.

## ▶️ How to Run
1. Open the project in Eclipse IDE
2. Compile the project
3. Run the main class
4. Follow the instructions on screen
